# calebfernandes

Personal website scaffold for Caleb Fernandes, built as a static GitHub Pages site.

## Files

- `index.html` - page structure and content
- `styles.css` - responsive visual design
- `script.js` - small navigation and reveal interactions
- `assets/calebfernandes-hero.png` - generated hero image used on the first screen

Open `index.html` in a browser, or serve the folder with any static server.
